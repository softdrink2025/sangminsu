# 🎶 영포티파이 🎶

## 서비스 개요
'영포티파이'는 사용자 맞춤형 음악 추천 서비스를 제공합니다. 사용자는 다른 사람들과 소통하거나 AI 기반의 추천 시스템을 통해 자신만의 음악을 추천받을 수 있습니다. 또한, 다른 사용자에게 추천을 하거나 유용한 기능들을 활용하여 더 나은 음악 경험을 제공합니다.

## 주요 기능
- 🔐 **로그인/회원가입**: 사용자는 계정을 생성하고 로그인하여 개인화된 서비스를 이용할 수 있습니다.
- 📝 **게시판 CRUD**: 음악에 대한 게시물을 작성하고 관리할 수 있는 게시판 기능을 제공합니다.
- 💬 **댓글 기능**: 다른 사용자의 게시물에 댓글을 달 수 있어 소통할 수 있습니다.
- 👤 **프로필 페이지**: 사용자는 자신의 프로필을 확인하고, 수정할 수 있습니다.
- 🎧 **음악 추천 기능**: AI와 사용자 간의 상호작용을 통해 맞춤형 음악을 추천받을 수 있습니다.

## 기술 스택
- 🖥️ **Django**: 서버 사이드 웹 프레임워크
- 🗄️ **SQLite**: 데이터베이스 관리
- 🎨 **Bootstrap**: 사용자 인터페이스 디자인
- 🎵 **Spotify API**: 음악 추천 및 스트리밍 서비스 연동
- 📺 **YouTube API**: 뮤직 비디오 제공 및 추천 서비스

## 서비스 분석 요약
영포티파이는 사용자의 데이터를 기반으로 맞춤형 음악 추천 서비스를 제공하며, 다양한 기능을 통해 사용자들이 음악을 발견하고 공유할 수 있게 합니다.

## 변경 및 추가 기능
- 🎥 **뮤직 비디오 링크 제공**: 사용자가 추천받은 음악에 대해 뮤직 비디오 링크를 제공합니다.
- 💰 **포인트 제공**: 사용자는 활동에 따라 포인트를 적립하고 이를 서비스 내에서 사용할 수 있습니다.
- 🎶 **플레이 리스트 & 선호 음악 제공**: 사용자는 자신이 좋아하는 음악과 플레이리스트를 관리할 수 있습니다.
- 🎤 **노래 제목 보고 앨범 명 맞추기**: 음악을 제목만 보고 앨범명을 맞추는 퀴즈 기능을 추가합니다.
- 🎼 **노래 가사 & 흥얼거리면 노래 검색 ('노래 찾아주세요' 게시판)**: 사용자가 흥얼거리거나 가사 일부를 입력하여 노래를 찾을 수 있는 게시판 기능을 추가합니다.

## 주요 페이지 구성
1. 🏠 **메인 페이지**: 서비스의 기본적인 소개와 기능을 안내하는 페이지입니다.
2. 🤖 **AI를 활용한 음악 추천 페이지**: 사용자가 AI를 통해 맞춤형 음악을 추천받는 페이지입니다.
3. 🎤 **사용자들 간의 음악 추천 게시판**: 사용자들이 서로 추천한 음악을 게시하고 의견을 나눌 수 있는 페이지입니다.
4. 🔍 **'노래 찾아주세요' 게시판**: 사용자가 노래 제목이나 가사로 노래를 찾을 수 있는 게시판입니다.
5. 🔑 **로그인/회원가입 페이지**: 사용자가 계정을 만들고 로그인할 수 있는 페이지입니다.
6. 👤 **프로필 페이지**: 사용자가 자신의 프로필을 확인하고 수정할 수 있는 페이지입니다.
